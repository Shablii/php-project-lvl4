##Что бы установить необходимо:
- скачать проект: 
> git clone git@github.com:Shablii/php-project-lvl4.git
- перейти в папку с проектом:
> cd php-project-lvl4
- установить:
> make setup

Что бы локально запустить используйте  в корне проекта
> make start

###Требования:
PHP 8/node 6.14.15
